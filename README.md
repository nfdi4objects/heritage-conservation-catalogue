# Heritage Conservation Catalog
Kuratierte Sammlung von Online-Ressourcen für die Konservierungswissenschaft sowie die Denkmalpflege/ den Denkmalschutz. Enthält u.a. Vokabulare, Ontologien, Datenmodelle und Fachtexte und soll als Recherchewerkzeug eine Übersicht über die vielfältige Datenlandschaft bieten.
